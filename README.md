# Disease Prediction System using Machine Learning

## Overview
A Flask-based web application that predicts potential diseases from selected symptoms using Machine Learning models.

## Features
- Symptom-based disease prediction
- KNN and SVM models
- Confidence scores for predictions
- Interactive web interface
- Fast and lightweight deployment

## Technologies Used
- Python
- Flask
- Scikit-learn
- NumPy
- HTML
- CSS

## Project Structure
```
app.py
model.py
models/
templates/
static/
archive/
```

## Machine Learning Models
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)

## Installation

```bash
pip install -r requirements.txt
python app.py
```
